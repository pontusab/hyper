# @usehyper/openapi-arktype

SchemaConverter for ArkType to @usehyper/openapi.

## Install

```bash
bun add @usehyper/openapi-arktype
```

## Usage

```ts
import { arktypeConverter } from "@usehyper/openapi-arktype"
```
