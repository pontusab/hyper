# @hyper/openapi-arktype

SchemaConverter for ArkType to @hyper/openapi.

## Install

```bash
bun add @hyper/openapi-arktype
```

## Usage

```ts
import { arktypeConverter } from "@hyper/openapi-arktype"
```
