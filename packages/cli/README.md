# @hyper/cli

Hyper CLI — init, dev, build, typecheck, env, routes.

## Install

```bash
bun add @hyper/cli
```

## Usage

```ts
bun x hyper dev
bun x hyper test --fuzz --types
bun x hyper security --check
bun x hyper bench --tests
```
