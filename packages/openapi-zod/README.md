# @hyper/openapi-zod

SchemaConverter for Zod (v3 + v4) to @hyper/openapi.

## Install

```bash
bun add @hyper/openapi-zod
```

## Usage

```ts
import { zodConverter } from "@hyper/openapi-zod"
```
