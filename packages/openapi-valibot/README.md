# @usehyper/openapi-valibot

SchemaConverter for Valibot to @usehyper/openapi.

## Install

```bash
bun add @usehyper/openapi-valibot
```

## Usage

```ts
import { valibotConverter } from "@usehyper/openapi-valibot"
```
