# @hyper/msgpack

MessagePack wire format for Hyper — content-negotiated encode/decode.

## Install

```bash
bun add @hyper/msgpack
```

## Usage

```ts
import { msgpack } from "@hyper/msgpack"
app({ use: [msgpack()] })
```
